# netflix_title_analysis

#Project Overview

This project analyzes the Netflix Titles dataset to gain insights into the content available on Netflix. The focus is on understanding trends in content over time, top contributors, and the global distribution of movies and TV shows.

The dataset contains information about each title, such as directors, cast, countries, release year, ratings, and genres.

#Dataset Description

The dataset netflix_titles.csv includes the following columns:
 Column Name    | Description                                            
 show_id        | Unique identifier for each title                       
 type           | Type of content (Movie or TV Show)                      
 title          | Name of the movie or TV show                            
 director       | Director(s) of the content                              
 cast           | Main cast members                                       
 country        | Country of production                                   
 date_added     | Date the title was added to Netflix                     
 lease_year     | Original release year                                   
 rating         | Content rating (PG, TV-MA, R, etc.)                     
 duration       | Duration of the movie or number of seasons for TV shows 
 listed_in      | Genre(s)                                                
 description    | Brief description of the content     

# Project Objectives

1.Analyze content Movies and TV shows to understand Netflix’s expansion.
2.Identify top genres, directors, and actors contributing to Netflix content.
3.Explore country-wise production and distribution of content.
4.Create interactive dashboards and visualizations using Power BI.

#Data Cleaning & Preparation

1.Removed duplicate records.
2.Handled missing values in director, cast, and country columns.
3.Standardized content ratings and genre categories.
4.Converted date_added to a proper date format.
5.Performed aggregations and calculations using SQL queries.

#Analysis & Insights

Key analyses include:
1.Content Growth Over Year – Number of movies and TV shows added each year.
2.Top 10 Countries Producing Content – Countries contributing the most content.
3.Most Frequent Genres – Popular content categories on Netflix.
4.Top Directors and Actors – Contributors with the most titles.
5.Content Rating Distribution – Number of titles by rating (PG, TV-MA, R, etc.).

#Tools & Technologies

- Data Cleaning & Analysis: MySQL / SQL Server
- Data Visualization & Dashboarding: Power BI
